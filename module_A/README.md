## Состав модуля
- docs – спецификации архитектуры, паспорта витрин.
- diagrams – диаграмма потоков и схема медальонных слоёв.
- reports – итоговый отчёт в .ipynb и .html.
- schema - физическая модель.

## Как проверять
1. Откройте reports/Report_A.html – полный отчёт.
2. Для деталей:
   - docs/architecture_overview.md – выбор стека, слои Bronze/Silver/Gold.
   - docs/mart_specifications.md – 7 витрин.
   - schema/data_model_physical.sql – DDL для ClickHouse.
3. Диаграммы в diagrams (открываются в draw.io).
