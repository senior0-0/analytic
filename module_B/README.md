Модуль В реализует:
1. Интерактивный дашборд - для визуализации транспортных потоков и прогнозов
2. Модель машинного обучения - для прогнозирования загруженности дорог

Дашборд подключается к данным из Модуля Б (через ClickHouse) и отображает:
- Текущую ситуацию на дорогах
- Прогнозы на 30, 60 и 120 минут
- Анализ исторических данных
- Структуру транспортного потока

## Системные требования
- Python 3.8+
- Docker и Docker Compose 
- 4+ GB RAM
- streamlit

## Python зависимости
pip install streamlit pandas numpy matplotlib seaborn scikit-learn clickhouse-driver
